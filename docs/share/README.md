微信实例

**可直接传入二维码url/base64或者传入url自动生成qrcode**

```js
wechatData:{
  url:'https://leafiy.github.io/gelatin-ui/',
},
wechatData2:{
  qrcode:'https://user-images.githubusercontent.com/870449/49635434-aea69500-fa3a-11e8-9a2b-e70253f450df.png',
}
```
